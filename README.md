# Big Black Puppet Independent

Standalone static website for **Big Black Puppet — Sasha Persholja**.

This version is independent from Vercel and can be hosted on:

- GitHub Pages
- Cloudflare Pages
- Hostinger / HostGator / any cPanel hosting
- Apache / Nginx
- Any static web server

## Required local assets

The page expects these files:

```text
assets/video/big-black-puppet.mp4
assets/images/Sasha Persholja Live.png
```

Upload those files to the same paths in this repository for the full visual version.

## Entry point

Open:

```text
index.html
```
